First Time Setup

To use this app, you'll need a GitHub Personal Access Token (PAT). Here's how to get one:

  1. Go to GitHub Fine-grained Tokens
  2. Click "Generate new token"
  3. Set a token name (e.g., "Futel Operator Log")
  4. Set expiration (recommend: 90 days)
  5. Under "Repository access" select "Only select repositories"
  6. Choose the "myklemykle/logs" repository
  7. Under "Repository permissions", expand "Contents" and set it to "Read and write"
  8. Click "Generate token"
  9. Copy your new token (it starts with 'github_pat_')
⚠️ Save your token somewhere safe! You won't be able to see it again after leaving the page.
